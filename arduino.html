Arduino Project

Description:

My Arduino project is a traffic light system where the system goes green to go, yellow to slow down, and red to stop. When it gets to red blue lights up to give a signal so a pedestrian can walk across.

Process:

It took me a while to get this project down after all the errors and complications but when I found out what I was doing wrong It was way easier. Since the first tutorial I was using wasnt as clear to me I used a different traffic light tutorial to help me.

Reflection:

One thing I learned is how to use the Arduino board and make lights turn on by doing a series of coding and wiring. One thing I would do different is learn a bit more to add something more complex onto the basic tutorial.

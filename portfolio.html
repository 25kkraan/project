Video Game Design

Description:

My game is about a monkey trying to get food. In the game you have to catch bananas in order to keep the monkey from starving and If you dont get enough bananas you lose.

Process:

The process that I had to go through was finding a character and a object I thought of a cup and a ball but I found that had no backstory so I decided to pick a monkey catching a banana and thats how my game got made.

Reflection:

One thing I learned is that it is pretty easy to make a catch game on scratch and it is pretty fun. One thing that I would do differently is probally add more things to make more backstory.
